# customcursors
Customize cursors. No one created a tool about it so i've made it

# Requirements

Homebrew Channel

# Cursors

`starwars` = Star Wars cursors probably LG did a collab
`wee` = Blue cursor similar to magenta cursor. Only supports medium sized cursor not small, large.
`default` = Default cursor, depends on the files.
`beanbird` = Beanbird cursor
`webos` = Netcast cursor (coming soon)
`miracast` = Screen share cursor (coming soon)
`screenshare` = Miracast cursor (coming soon)
`netcast` = Prototype cursor imo (coming soon)
